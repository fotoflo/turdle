# Wordle Solver by Alex Miller <fotoflo@gmail.com>

a fun and simple open source solver for wordle

to develop:
```
cd app
npm start
```

to build:
```
cd app 
npm run build
cd ..
git add .
git commit "you have to commit to origin main"
git push origin main
```

to run tests (for the Hint Helpers)

```
cd app
npm test
```
